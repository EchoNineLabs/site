List of all commands and permissions provided by Kite.

<br>

<div class="commands">

| Command                 | Permission   | Description                          |
| :---------------------- | :----------- | :----------------------------------- |
| `/kite list (script)`   | `kite.admin` | Shows list of all available scripts. |
| `/kite load (script)`   | `kite.admin` | Loads a script by name.              |
| `/kite unload (script)` | `kite.admin` | Unloads a loaded script.             |
| `/kite reload (script)` | `kite.admin` | Realoads a script.                   |

</div>

{{< callout type="warning" >}}
  Please be cautious and give access to these commands only to people you trust.
{{< /callout >}}
